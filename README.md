## Open-Source MATLAB Equivalents

- [GNU Octave](https://www.gnu.org/software/octave/) – Open-source MATLAB-compatible environment.
- [SciLab](https://www.scilab.org/) – Numerical computation alternative.
- [NumPy](https://numpy.org/) – Python numerical computing (frequently used as MATLAB alternative).
- [MATPLOTLIB](https://matplotlib.org/) – Visualization library akin to MATLAB plotting.
- [SciPy](https://scipy.org/) – Scientific computing library for Python.

## External Libraries & Utilities

- [matlab2tikz](https://github.com/matlab2tikz/matlab2tikz) – Convert MATLAB figures to TikZ/PGFPlots.
- [jsonlab](https://github.com/fangq/jsonlab) – JSON encoder/decoder for MATLAB.
- [YALMIP](https://github.com/yalmip/YALMIP) – Optimization modeling language for MATLAB.
- [CVX](http://cvxr.com/cvx/) – Convex optimization modeling framework.
- [MOSEK MATLAB API](https://www.mosek.com/) – Numerical optimization solver.
